# Dataset Attribution

## MC_Maze Dataset

**Source:** DANDI Archive  
**Dandiset ID:** 000140 (MC_Maze_Small)  
**URL:** https://dandiarchive.org/dandiset/000140/draft  
**License:** CC-BY-4.0 (Creative Commons Attribution 4.0 International)

### Description
Macaque primary motor and dorsal premotor cortex spiking activity during delayed reaching. This dataset contains sorted unit spiking times and behavioral data from a macaque performing a delayed reaching task with obstructing barriers.
